Create HTML from an array of objects.

You are given a constant backpackObjectArray containing an array of two objects and the function createBackpackMarkup() which returns the constant content containing a template literal with HTML output.
 
Your task: 
  • Iterate through each item in the backpackObjectArray.
  • Create a new <article> with the id matching the id field in the backpack object.
  • Use the backpackAge() function to calculate the backpack age and add it in place of the purchase date (dateAcquired).
  • Populate and return content with a HTML structure matching the example below:
<article id="pack01">
	      <h1>Everyday Backpack</h1>
	      <ul>
              <li>Volume:<span> 30l</span></li>
              <li>Color:<span> grey</span></li>
              <li>Age:<span> 1594 days old</span></li>
              <li>Number of pockets:<span> 15</span></li>
	      </ul>
	      </article>

Parameters

document: The `document` property of the project DOM. 

Result

Serialized HTML fragment describing two <article> elements populated with the data from backpackObjectArray.

Constraints

  • The returned content HTML structure must match the stated example.
